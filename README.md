#  GWAS_Soybean_Pipeline

Данный репозиторий содержит решение тестового задания по Геномному Анализу Ассоциаций (GWAS) урожайности сои.

## 01. Решение: GWAS-анализ (Jupyter Notebook)

* **Файл:** `01_GWAS_Notebook/GWAS_Analysis_Result.ipynb`
* **Содержимое:** Пошаговый интерактивный анализ GWAS с использованием OLS-регрессии (Ordinary Least Squares), скорректированной на структуру популяции (PCA).
* **Результаты:** Включает визуализацию результатов (Manhattan Plot и QQ-Plot) и интерпретацию.

##  02. Решение: Автоматизация Пайплайна (Snakemake)

Этот раздел содержит полностью автоматизированный рабочий процесс для повторения GWAS, используя систему управления пайплайнами **Snakemake**.

###  Требования для Запуска

1.  **Программы (CLI):** `Snakemake`, `vcftools`.
2.  **Python-библиотеки:** `scikit-allel`, `scikit-learn`, `numpy`, `pandas`, `matplotlib`, `statsmodels`, `scipy`.

## Рекомендованный метод запуска:
 Использование явного указания конечной цели (gwas_results.qq.png) гарантирует корректное выполнение всех шагов пайплайна.
`snakemake --cores 1 gwas_results.qq.png`
 Из-за конфликтов кодировки/невидимых артефактов, возникших при переносе файлов между операционными системами, стандартный запуск (snakemake --cores 1) не всегда может корректно прочитать rule all. Для обеспечения полной работоспособности пайплайн запускается с явным указанием конечного файла.
